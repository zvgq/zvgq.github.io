Here you will find the high level plan for the ZVGQ revamp project. As the plan develops, we will be adding more to this page, so be sure to check back once and a while.

Phase 0 - Planning (We are here today)

Goal: Make the blueprint and strategy to move forward!
  • Build a development team
  • Outline the general architecture for the ZVGQ application
  • Determine the core technologies that drive ZVGQ-API and ZVGQ-WEB projects
  • Do some experimentation with tech and setup

Phase 1 - ZVGQ-API

Goal: Create the ZVGQ-API application and build a basic "new" ZVGQ site that uses it.
  • Build a RESTful API application to deliver ZVGQ content
  • Setup a basic website and shows games, quotes, and the media associated with each item
  • Develop a method for content administrators to administer content

Phase 2 - ZVGQ-WEB

Goal: Flesh out the ZVGQ-WEB application to make it what is should be
  • Implement a simple ZVGQ brand and style
  • Add more features that are identified by the community of users
  • Integrate other ZVGQ communication channels (e.g. Facebook, Twitter, etc...)

Phase 3 - ZVGQ NEXT

Goal: Not sure yet, but we'll get there!
